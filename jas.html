What Is WebDesign?

  • HTML

HTML stands for Hyper Text Markup Language. It is used on the frontend and gives the structure to the webpage which you can style using CSS and make interactive using JavaScript.

  1. The DOCTYPE html tag declaration defines that this document is an HTML5 document.

  2. The html tag element is the root element of an HTML page.
  3. The head tag element contain meta information about the HTML pag.
  4. The title tag element specifies a title for the HTML page(which is shown in the browser's title bar or in the page's tab )
  5. The body tagelement define the document's body, and is a container for all the visible contents, such as headings, paragraphs,images,hyperlinks,tables,lists,etc.
  6. The h1 tag element definesa large heading
  7. The p tag element defines a paragraph

  • CSS

  1. CSS stands for Cascading Style Sheets
  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other mediea
  3. CSS saves a lot of work. It can cantrol the layout of multiple web pages all at once
  4. External stylesheets are stored in CSS files

  • External CSS

  • Internal CSS

  • Inline CSS
